# Node Quickstart

A very simple Node.js example.

## Installation

Mac

```
brew install node
```

Linux

```
cd ~/Downloads
wget -O nodejs.tar.xz https://nodejs.org/dist/v7.0.0/node-v7.0.0-linux-x64.tar.xz
sudo tar -C /usr/local --strip-components 1 -xJf nodejs.tar.xz
```

## Upgrading

Mac

```
brew upgrade node
```
